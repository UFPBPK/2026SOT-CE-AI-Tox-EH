This Python code is a practice exercise for the CE course. It is a simplified version of the code from my recent publication in Environmental Pollution (https://doi.org/10.1016/j.envpol.2025.127519).

Key Points

- Focuses on respiratory carcinogenicity only. The original publication includes five organ-specific carcinogenicity models; this practice uses only one.
- Uses RDKit and MACCS as example tools for molecular representation and feature generation.
- Written in Python and intended to be executed in a Jupyter Notebook environment.
- Ensure that all required Python packages are installed. Please run the notebook cell by cell.
